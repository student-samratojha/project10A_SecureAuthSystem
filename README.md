Project10A - Secure Access System 🔐
Project10A is a robust Full-Stack Authentication and User Management system built with security and scalability in mind. It features a complete Admin-User workflow, including manual account approvals and detailed activity tracking.

🚀 Features
Security
JWT Protected: Secure communication using JSON Web Tokens.

Encrypted Q&A: Password recovery via encrypted security questions to ensure data privacy.

Role-Based Access: Distinct workflows for regular Users and Administrators.

Admin Capabilities
User Approval System: New registrations remain "Pending" until an Admin manually verifies them.

User Management: Ability to revoke access or delete users from the database.

System Audit Logs: A comprehensive log of every action (Logins, Registers, Approvals, Logouts) with timestamps and IP addresses.

User Experience
Personalized Dashboard: Users can view their account status and recent activity.

Profile Management: Update personal details, bio, and profile pictures seamlessly.

Secure Recovery: Easy password reset flow using pre-configured security questions.

🛠️ Tech Stack
Frontend: React.js / Next.js (Tailwind CSS for styling)

Backend: Node.js & Express.js

Database: MongoDB

Authentication: JWT (JSON Web Tokens) & Bcrypt for hashing

📸 Preview
Login Page: Clean UI with "Forgot Password" and registration links.

Admin Console: Real-time stats of pending, verified, and deleted users.

Audit Table: Detailed tracking of system-wide events.
